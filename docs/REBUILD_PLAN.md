# پلن بازسازی ۱.۰ — AirsoftPlatform

> این سند مرجع فاز «بازسازی ۱.۰» است. تسک‌های کامل با معیار پذیرش در Notion (پروژه «پلتفرم ایرسافت — پلن کامل تا پروداکشن»، تسک‌های ۵۶ تا ۶۷) نگهداری می‌شوند.

## وضعیت فعلی (تحلیل)

- **بک‌اند:** FastAPI (Python 3.12) + SQLAlchemy 2 + PostgreSQL 16 — روترها: auth، catalog، bookings، events، leagues، shop، payments، admin، uploads. بدون Alembic (فقط create_all)، بدون تست.
- **فرانت:** Next.js 14 (app router) + Tailwind — صفحات: home، fields، events، booking، shop، leagues، gallery، rules، login/register، profile، payment، admin. فقط ۲ کامپوننت مشترک (Navbar، Footer)، بدون دیزاین‌سیستم و انیمیشن.
- **زیرساخت:** Docker Compose (db / backend / frontend / nginx). پروژه در محیط آفلاین ساخته شده و هنوز بیلد کامل واقعی نگرفته.
- ⚠️ **مشکل امنیتی:** فایل `.env` با `SECRET_KEY` و رمز دیتابیس در گیت کامیت شده بود (در همین PR حذف شد). **همه secretها باید rotate شوند** چون در تاریخچه گیت باقی مانده‌اند.

## اهداف بازسازی ۱.۰

1. **تمیزکاری و اصلاح مسیرها** — همه روت‌ها و فلوهای فعلی بدون خطا کار کنند (تسک ۵۶، ۵۷)
2. **UI مدرن با حس Airsoft** — دیزاین‌سیستم تاکتیکال دارک + Framer Motion + افکت‌های HUD (تسک ۵۸، ۵۹)
3. **Armory + Loadout Builder** — نمایش گان‌ها با عکس واقعی، انتخاب تجهیزات به ازای هر گان به سبک Gunsmith (تسک ۶۰ تا ۶۳)
4. **سانس‌های جدید** — ۱۰ سانس روزانه × ۷ روز طبق جدول مجموعه (SessionTemplate + تولید خودکار) (تسک ۶۴، ۶۶)
5. **سیدر پروداکشن** — ماژولار، idempotent، با عکس‌های واقعی گان‌ها و تجهیزات (تسک ۶۱، ۶۵)
6. **QA و آماده‌سازی پروداکشن** (تسک ۶۷)
